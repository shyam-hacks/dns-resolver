# DNS Resolver 🔍

A simple Python tool to resolve a domain name to its IPv4 or IPv6 address using Python's `socket` module.

## 🚀 Features

- Resolve IPv4 addresses using `gethostbyname()`
- Resolve IPv6 addresses using `getaddrinfo()`
- User-friendly CLI input

## 🛠️ Technologies Used

- Python 3.x
- `socket` module (built-in)

## 📦 Installation

```bash
git clone https://github.com/YOUR-USERNAME/dns-resolver.git
cd dns-resolver
python3 dns_resolver.py
